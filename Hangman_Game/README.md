# CodeAlpha_HangmanGame

## Internship Task
This project was developed as part of the **CodeAlpha Python Programming Internship**.

## Description
A simple text-based Hangman game built in Python. The player tries to guess a hidden word one letter at a time. The game ends when the player either guesses the full word or runs out of attempts (6 incorrect guesses allowed).

## Features
- Random word selection from a predefined word list
- Tracks correct and incorrect guesses
- Displays remaining attempts
- Option to replay after a game ends

## Concepts Used
- `random` module
- `while` loops
- `if-else` conditionals
- Strings and lists
- Basic input validation

## How to Run
```bash
python hangman_game.py
```

```

Wrong guesses left: 6

Good guess! 'p' is in the word.
```
