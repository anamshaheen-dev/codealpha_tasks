# CodeAlpha_StockPortfolioTracker

## Internship Task
This project was developed as part of the **CodeAlpha Python Programming Internship**.

## Description
A simple command-line stock portfolio tracker that calculates the total investment value based on user-entered stock quantities and a predefined dictionary of stock prices.

## Features
- Predefined dictionary of stock prices (AAPL, TSLA, GOOGL, MSFT, AMZN)
- Accepts multiple stocks and quantities from the user
- Calculates total investment value
- Displays a breakdown per stock
- Optionally saves the summary to a text file (`portfolio_summary.txt`)

## Concepts Used
- Dictionaries
- Loops and conditionals
- Basic arithmetic
- File handling (optional save feature)
- Input validation

## How to Run
```bash
python stock_portfolio_tracker.py
```

## Sample Output
```
AAPL: 10 shares x $180 = $1800
TSLA: 5 shares x $250 = $1250
----------------------------------------
Total Investment: $3050
```
