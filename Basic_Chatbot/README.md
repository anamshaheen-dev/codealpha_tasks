# CodeAlpha_BasicChatbot

## Internship Task
This project was developed as part of the **CodeAlpha Python Programming Internship**.

## Description
A simple rule-based chatbot built in Python. The chatbot responds to predefined inputs such as greetings, simple questions, and farewells using `if-elif` logic.

## Features
- Responds to greetings (hello, hi, hey)
- Responds to basic questions (how are you, what is your name)
- Responds to thanks and goodbyes
- Falls back to a default response for unrecognized input
- Runs in a continuous loop until the user types "bye"

## Concepts Used
- Functions
- `if-elif` conditionals
- Loops
- String input/output handling

## How to Run
```bash
python basic_chatbot.py
```

## Sample Conversation
```
You: hello
Bot: Hi! How can I help you today?
You: how are you
Bot: I'm fine, thanks! How about you?
You: bye
Bot: Goodbye! Have a great day.
```
